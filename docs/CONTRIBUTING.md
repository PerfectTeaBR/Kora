# 😃 Contribuindo para o Kora
**Que bom que você quer ajudar!!**

Siga os passos para contribuir que é sucesso!

1. Issues:
   - Caso você encontre um problema no código, abra um Issue que nossa equipe irá resolver.

2. Pull Requests(PR)
   - Agora se você quer colocar a mão na massa e adicionar sua funcionalidade, você pode abrir um pull request, desde que a adição da sua funcionalidade não viole as regras e a política de privacidade do Github.
